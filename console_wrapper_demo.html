Additional Commands

The conosole API can be found here: - Console APIs: http://getfirebug.com/wiki/index.php/Console_API
This console wrapper allows you to set levels:
$.console.setLevel(pLevel);
Available levels are: off, exception, error, warn, info, log, debug. Log and debug are equivalent.
To view current debug level: $.console.getLevel();

When used in APEX applications that are running in APEX Debug mode, level is automatically set to "log".
